# 博客部署

# 说明

该博客forkxugaoyi/vuepress-theme-vdoing (opens new window).替换原有文章内容和个人信息形成。并且基于Github Pages部署，图片基于https://picx.xpoet.cn/ (opens new window)上传,集成Gitalk参考资料 (opens new window)。

# 一、前提准备

# 1.fork项目

访问vuepress-theme-vdoing (opens new window),完成项目创建。

# 2配置项目Github pages

进入项目仓库主页。

注：有一个gh_page的分支，发布的页面会提交在这个页面。需要在配置github pages时注意。 进入Setting配置页面,完成Pages分支的配置

# 3.添加项目secret

参考：部署 (opens new window)

首先创建具有仓库访问权限的access token (opens new window)。

然后配置到项目的Setting>General>Security>Secrets>Actions添加键值对，name默认ACCESS_TOKEN。

见项目文件.github>workflows>ci.yml文件，会将键值对取出作为执行deploy.sh的参数。

# 4.配置GitTalk
